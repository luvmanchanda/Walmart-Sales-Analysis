# Walmart Sales Analysis using Python & MySQL

## Project Overview

This project demonstrates an end-to-end data analytics workflow using the Walmart Sales dataset from Kaggle. The objective was to transform raw sales data into meaningful business insights by performing data cleaning and preprocessing in Python, storing the processed data in MySQL, and solving real-world business problems using SQL queries.

## Dataset

* **Source:** Kaggle – Walmart Sales Dataset
* **Tools Used:** Python, Pandas, Google Colab, MySQL, SQL

## Project Workflow

### 1. Data Collection

* Downloaded the Walmart Sales dataset from Kaggle.
* Imported the dataset into Google Colab using Pandas.

### 2. Data Cleaning & Preprocessing

* Inspected the dataset for missing values and inconsistencies.
* Removed duplicate records and standardized data formats.
* Converted columns to appropriate data types.
* Created additional features required for analysis.
* Exported the cleaned dataset for database integration.

### 3. Database Integration

* Created a MySQL database and table schema.
* Imported the cleaned dataset into MySQL.
* Verified data integrity and prepared the database for analysis.

### 4. Business Problem Solving with SQL

Using SQL, the following business questions were answered:

* Analyzed payment methods and sales performance.
* Identified the highest-rated product category in each branch.
* Determined the busiest day of the week for every branch.
* Calculated the total quantity sold through each payment method.
* Compared category ratings across different cities.
* Calculated total profit generated by each product category.
* Identified the most commonly used payment method in every branch.
* Analyzed transaction distribution across Morning, Afternoon, and Evening shifts.

## Key Learnings

* Data cleaning and preprocessing using Python (Pandas).
* Importing datasets into MySQL databases.
* Writing SQL queries using filtering, aggregation, grouping, sorting, CASE statements, subqueries, CTEs, and window functions.
* Converting raw retail data into actionable business insights for decision-making.
